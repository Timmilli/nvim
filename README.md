# Neovim config

From [Kickstart](https://github.com/nvim-lua/kickstart.nvim), I'm using the following plugins with [Lazy](https://github.com/folke/lazydev.nvim) :

- [autocompletion](https://github.com/saghen/blink.cmp.git)
- [autopairs](https://github.com/windwp/nvim-autopairs)
- [catpuccin](https://github.com/catppuccin/nvim)
- [conform](https://github.com/stevearc/conform.nvim)
- [gitsigns](https://github.com/lewis6991/gitsigns.nvim)
- [indent_line](https://github.com/lukas-reineke/indent-blankline.nvim)
- [lint](https://github.com/mfussenegger/nvim-lint)
- [lspconfig](https://github.com/neovim/nvim-lspconfig) & [Mason](https://github.com/mason-org/mason.nvim)
- [markdown-preview](https://github.com/iamcco/markdown-preview.nvim)
- [neo-tree](https://github.com/nvim-neo-tree/neo-tree.nvim)
- [precommit](https://github.com/Ttibsi/pre-commit.nvim)
- [telescope](https://github.com/nvim-telescope/telescope.nvim)
- [treesiter](https://github.com/nvim-treesitter/nvim-treesitter)
- [vimtex](https://github.com/lervag/vimtex)
- [whichkey](https://github.com/folke/which-key.nvim)
